reload()


定义和用法
reload() 方法用于重新加载当前文档。
如果该方法没有规定参数，或者参数是 false，它就会用 HTTP 头 If-Modified-Since
来检测服务器上的文档是否已改变。如果文档已改变，reload() 会再次下载该文档。如果文档未改
变，则该方法将从缓存中装载文档。这与用户单击浏览器的刷新按钮的效果是完全一样的。

如果把该方法的参数设置为 true，那么无论文档的最后修改日期是什么，它都会绕过缓存，从服务
器上重新下载该文档。这与用户在单击浏览器的刷新按钮时按住 Shift健的效果是完全一样。
